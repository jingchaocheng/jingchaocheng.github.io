Vim「更现代」的使用方式

发表于 2020-06-13 | 分类于 编辑器

背景介绍

Vim 是从 vi 发展出来的一个文本编辑器。和 Emacs 并列成为类 Unix 系统用户最喜欢的编辑器。

Vim 提供了一系列很强大的编辑文本的操作方式，可以使用纯键盘完成编辑文本操作。再搭配许多插件可以配置成一个 IDE。但是 Vim 有着一个比较陡峭的学习曲线，并且不是开箱即用的。

如果没有特定的开发环境限制，可以使用

更漂亮易上手的 GUI（Visual Studio Code） + 更流畅的编辑体验 （Vim）

来提高编码效率，各取所长。

如今市面上几乎所有的 IDE 和编辑器都提供有 Vim 的相关插件。例如我所体验过的

  • Sublime Text - Vintageous、Six
  • Visual Studio - VsVim
  • Visual Studio Code - Vim

本文的所有操作都基于 Visual Studio Code + Vim 插件 演示。(虽然我更喜欢 Sublime…)
下文出现的 Vim 均代表 Vim 插件

Vim 模式

normal 模式

文本编辑不仅仅只是输入，还有删除，选择，移动等操作。normal 模式下有很多命令可以实现

normal 表示自然、放松的状态，Vim设计之初就认为人们使用编辑器时其实只有小部分时间是在真正进行编辑操作，绝大多数时间会用于思考、阅读，以及在文件已有内容中穿梭浏览。

  • normal 无法输入
  • insert
  • :cmd 命令模式
  • v visual 可视化模式

Visual 模式

  • normal 模式下使用 v 进入 visual 选择
  • V 选择行
  • Ctrl + v 块选择
    o 切换方向

d 删除

insert 模式

终端也可以使用
ctrl h 删除上一个字符
ctrl w 删除上一个单词
ctrl u 删除当前行

ctrl a start
ctrl e end
ctrl b 前
ctrl f 后

normal -> insert

i - insert
a - append
o - open a line below

I - insert before line
A - append after line
O - open a line above

gi 最后一次编辑并进入插入模式的地方

Chrome 插件 Vimium

Vimium 提供了在浏览器上使用 Vim 快捷键操作。

j 向下移动网页
k 向上移动网页
gg 移动到网页顶部
G 移动到网页底部

J、gT向左转一格
K，Gt向右转一圈
g0 进入第一个标签
g$ 转到最后一个标签
t 新建标签页
x 关闭当前标签
X还原已关闭的标签页(即解除’x’命令)
